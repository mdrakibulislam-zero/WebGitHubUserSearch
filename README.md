## Web GitHub User Search:
This is a simple GitHub user informations finder program based on Web which made with HTML, CSS and JavaScript. It's able to search GitHub user profiles and informations by username.

## What Have I Done:
I've made this GitHub user search program using HTML, CSS and JavaScript. I've used axios-0.21.0.js for made this program.

Axios is a Javascript library used to make HTTP requests from node.js or XMLHttpRequests from the browser and it supports the Promise API that is native to JS ES6. It can be used intercept HTTP requests and responses and enables client-side protection against XSRF. It also has the ability to cancel requests.
